# Northwind Sample SQL Queries

This project contains sample queries using the Northwind sample database found here: 

    https://github.com/microsoft/sql-server-samples/tree/master/samples/databases/northwind-pubs

Here are the steps I took:

1. Installed Sql Server
2. Downloaded the DB file - instnwnd.sql
3. Uploaded the DB into SQL Server
4. Ran queries
5. Took screenshots
6. Uploaded screenshots into northwind-sql-queries > screenshots
